# Color System

Our color system provides a comprehensive set of semantic color tokens that adapt across multiple themes. Each color is defined using HSL values for maximum flexibility and consistency.

## Theme Overview

The design system includes four distinct themes:

- **Blurple** (Default) - A blue-purple color scheme
- **Purple** - A vibrant purple theme
- **Emerald** - A fresh green theme
- **Rose** - A warm pink theme

## Usage Guide

### CSS Variables

All colors are available as CSS custom properties:

```css
background-color: hsl(var(--background));
color: hsl(var(--foreground));
border-color: hsl(var(--border));
```

### Tailwind Classes

```jsx
<div className="bg-background text-foreground">
  <button className="bg-primary text-primary-foreground">Primary Button</button>
  <div className="bg-card text-card-foreground border-border border">
    Card Content
  </div>
</div>
```

### Theme Switching

Use the theme switcher component to switch the themes
the theme colors wil automatically switch by using the tailwind class method
make sure to use the same colors as defined in the global css file
